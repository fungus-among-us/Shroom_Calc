## Features

- **By Weight**: Calculate total compounds from material mass
- **Target Dose**: Calculate required material from desired dose
- Searchable specimen library with alkaloid concentrations
- Dosing reference ranges (microdose to heroic)
- JSON profile editor
- No analytics, all data stored locally

## Privacy

Single network request to download specimen profiles. No telemetry or data collection.
